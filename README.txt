# CRUD Spring Boot + React

## Descripción del Proyecto

Este proyecto fue desarrollado por estudiantes de Ingeniería en Informática de cuarto semestre. El propósito de este proyecto es aplicar la arquitectura MVC en un servicio web cuya finalidad estará enfocada en la compra y venta de artículos coleccionables.

## Características

- **Backend**: API RESTful con Spring Boot.
- **Frontend**: Interfaz de usuario con React.
- **Gestión de Datos**: CRUD completo (Crear, Leer, Actualizar, Eliminar).
- **Persistencia**: Base de datos MySQL.
- **Validación**: Validación de datos en formularios.


## Requisitos Previos

Asegúrate de tener instalados los siguientes componentes:

- [Node.js](https://nodejs.org/) (v14 o superior)
- [npm](https://www.npmjs.com/) o [yarn](https://yarnpkg.com/)
- [JDK 17](https://www.oracle.com/java/technologies/javase-jdk17-downloads.html) o superior
- [Maven](https://maven.apache.org/)
- [MySQL](https://dev.mysql.com/downloads/installer/)

1. Para ejecutar este proyecto es importante tener una Base de datos local en MySQL y ejecutar el Backend en Springboot el cual debe correr en el puerto 8080. Una vez ejecutado el Backend, se debe ejecutar el FrontEnd en React el cual debe estar corriendo en el puerto 3000.

## Uso

1. Accede a la aplicación en tu navegador o acceder directamente a http://localhost:3000/ en tu navegador
